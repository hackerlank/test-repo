Overview of the Administration Interface
 Previous Top Next

In order to access the Administration Interface, your user account must have administration privileges. If you do not have these privileges, please contact your system administrator to make the desired changes on your behalf.

The following functionalities are available to the Administrator(s):

·    View online users;
·    Log user off remotely;
·    Create or edit User Categories;
·    Create or edit User Accounts;
·    Create or edit Termbases and Projects;
·    Create or edit Fields;
·    Create or edit Input Models; and
·    Control Access Rights.

These functionalities are organised on the different tabs in the interface and each is discussed in detail in this manual.
