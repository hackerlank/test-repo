Exporting Terms
 Previous Top Next

To make the termbase that you have created with Autshumato TMS available in other software packages, you can export the termbanks, projects or terms.

To do so, click on the Export button to display the Export Dialog.  In this window, set the type of export as well as the export filter where applicable.

The 3 types of exports that are available are:

·    TAB

Setting this as the export filter, creates a tab separated document for use specifically with the Autshumato ITE. This is a normal text document with each line containing a term. The lines further contain the source language term and target language term separated by a tab. This type of export includes all of the target language terms and you therefore need not set the additional export filter.

·    ODT

This type generates the export document as an OpenDocument text file. If this option is selected the export filter will be displayed. In this section you have to select from the lists the record attribute fields, source and the target languages to include in the output. For the source and each target language the attribute fields also have to be selected. Only the fields and languages selected will be present in the exported document. You can select multiple fields in a list by holding the [Ctrl] keyboard button and clicking on the fields in the list.

·    TBX

This option generates the export document as a TermBase eXchange document, which is the standard for exporting, importing and transporting terms.

Click on the Generate button for the server to generate the export document. Once this has been done, the Download button will be displayed. Click on it to download the exported document.

The Autshumato TMS can export word lists (glossaries) to be used in the Autshumato ITE.
