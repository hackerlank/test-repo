The Autshumato TMS consists of three main pages:

·    The Welcome Page is the starting point of the Autshumato TMS. From there, you can log into the system and access the help file.

·    The Termbrowser Page is displayed once you have logged into the system. From this page, you can browse and search for terms, as well as create and edit records.

·    The Administration Interface Page is only for the Administrator and is where several aspects regarding the setup of the Autshumato TMS can be altered.

Each of the sections of these three pages is discussed in detail in this manual.
