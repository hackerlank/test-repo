The most frequently used technical terms in this manual and in the Autshumato ITE are defined below.
Termbank
A Termbank refers to a container of multiple Termbases.
Termbase
A database containing several records with terminology and related information.
Project
This divides a Termbase into smaller projects. A project is a sets of terms that all relate to the same specific context or topic.
Field
Any entity of a record.
Record Field
Fields that contain information about the record.
Index Field
Index fields are the fields that hold a term for a specific language. Thus each index field relates to a specific language.
Attribute Field
Field that contains information about an index field.
Input Model
A model that defines the record fields, index fields and attribute fields that may and must be present in a record. It also defines the access control for the individual fields.
Termbrowser
The tool that allows browsing of the records.
Administration Interface
The tool by which the Administrator can execute several tasks such as creating users, creating fields, creating input models, etc.
