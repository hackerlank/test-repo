Installing Autshumato TMS for individual users

The Autshumato TMS is a web-based application and can be accessed from any web browser. To start using the Autshumato TMS as an individual user, no installation is therefore necessary and you can navigate your web browser to the Start page (index.html).

The developers strongly recommend that the latest version of Mozilla Firefox be used as the web browser (see www.mozilla.com for more information). The latest version of the Firefox browser can be found here.


Installing Autshumato TMS on a server

A document is included with the source code of the Autshumato TMS. This document gives more detailed instructions regarding the installation of the Autshumato TMS on a server, as well as important information about the database, maintenance, security, etc.

This document can be found at /Documentation/Server Documentation.odt. It is recommended that the server administrator deal with this process.
