Browse Controls
 Previous Top Next

The Browse Controls can be used to browse through the available records. The controls are directly related to the source language, as the Browse Controls will go through the terms that are available in the source language in alphabetical order.

The picture below shows the Browse Controls for the default source language, English:



The first button on the Browse Controls bar is the GoTo Start button (marked 1 in the picture), which will navigate the Termbrowser to the first record available in the source language.

The last button on the Browse Controls bar is the GoTo End button (marked 2 in the picture), which will navigate the Termbrowser to the last record available in the source language.

The term before the one currently displayed in the database (marked 3 in the picture) is shown to the right of the GoTo Start button. Clicking on the term will navigate to that record. Similarly, the term after the current term (marked 4 in the picture) is displayed to the left of the GoTo End button.

In the middle of the Browse Controls, the Browse Box displays the source language term of the current record.

This box can also be used to quickly search for terms in the source language by simply typing the term that you want to search for in the Browse Box. If an exact match is found, the Termbrowser displays that record and updates the Browse Controls accordingly. If an exact match cannot be found, the Termbrowser navigates to the closest alphabetical match found and displays it.

For more detailed search functionalities, please see the relevant page in this manual.
