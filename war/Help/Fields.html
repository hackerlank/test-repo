Fields

The Fields tab enables you to create and edit fields. Fields are used to construct an input model, which can then be used to create, edit and display records.

Select an existing field from the drop-down list to edit, or click on the New Field... button to create a new field.

Once a field has been selected or created, the following Field Detail panel will be displayed:

The Field Detail panel has the following fields that need to be completed:

·    Field Name

In this field, you can enter the name of the field that is being edited or created.

·    Field Type

In this field, you can select one of the following field types:

·    Text Record Field;

·    Preset Record Field;

·    Index Field;

·    Text Attribute Field; or

·    Preset Attribute Field.

There is a distinction between Text and Preset fields. A Text field is a field that can contain generic text. A Preset Field can only contain one of a certain number of preset values that are set when creating the field. Fields with preset values are displayed as a drop-down list when creating or editing a Record. More information on the Preset fields is given below and the Preset Panel is discussed later in this section.

·    Field Data Type

In this field, you can specify the data type of the field that is being edited or created. You can select one of the following data types:

·    Plain text;

·    Integer;

·    Floating-point number;

·    HTML hyperlink; and

·    XML.

·    Maximum Length

In this field, you can specify the maximum number of characters or digits that may be entered in the field that is being edited or created.

·    Sort Index

In this field the sort index of the viewed field can be specified and modified.The sort index determines the display order of fields in a list.

Click the Save button to save the edited or newly created field. Click the Reset button to reset all the above fields to their default values. This will cause any data already entered to be lost.


Preset Panel

When a field with a preset data type has been selected for editing or created, the Preset Panel will be displayed.

The preset values for the field can be created or modified in this panel.

·    Existing Attribute

All the existing preset values for the field is listed in this drop-down box. The values can be selected for editing.

·    New Attribute

Clicking this button clears the text boxes and allows for the creation of a new preset value.

·    Attribute Value

This text box contains the value of the preset.

·    Attribute Key

The attribute key stores the preset value. Usually this is only the lowercase of the attribute value.

Click the Save button to save the edited or newly preset value. Click the Reset button to reset all the above values to their default values. This will cause any data already entered to be lost.
