Source and Target Languages
 Previous Top Next

It is possible to set the source and target languages used by the Autshumato TMS in the drop-down lists as shown in the picture below:

The drop-down boxes display all the index fields in the termbank; i.e. the languages that are available in the Autshumato TMS.

Selecting a language in the source language drop-down list:

·    changes the default language (English) to the language selected;

·    means that the Browse Controls will navigate through terms in the selected language;

·    ensures that search results will be displayed for terms found in the selected source language only;

·    acts as an additional filter that excludes all records that do not contain the chosen source language;

·    sorts the index fields so that the source language term will be the first index field to be displayed for each record.

The target language can only be set once the source language has been set.

By setting the target language, you specify the language of the term to be displayed (if available) immediately after the source language term. The other languages will still be displayed but will be placed alphabetically (sorted according to language name) after the source and target language terms. If there is no target language term for the selected record, the terms in other language will be displayed directly after the source language term.

Selecting the source and target languages eases the look-up, creation and editing of terms, as the user does not have scroll through the entire record to find the term in the desired language.
