Users

The User tab allows an administrator to create or edit individual user accounts.

Use the drop-down list to select an existing user account for editing. Click on the New User... button to create a new user. Certain details regarding each user are needed to create a profile. The table below lists the fields in the User tab that need to be completed:
First Name
In this field, you can view or edit an existing user’s first name or insert the first name of a new user.
Last Name
In this field, you can view or edit an existing user’s last name or insert the last name of a new user.
Username
In this field, you can either view the username for an existing account or edit the username. When you create a new user, you can insert a (unique) username in this field. The system automatically creates a unique username for a new user, but it can be replaced with a personalised username. The username should be at least 6 characters long and may only contain alphanumeric characters.
Password
The password can only be set in this field and can never be recovered. If a password is lost, assign a new password for the user account. The password must also be at least 6 characters long and may only contain alphanumeric characters.
Activated
This check box indicates whether the user account is active. Change the user’s status by checking or unchecking as appropriate.
Expiration Date
In this field, you can specify the date on which this user account will be de-activated.
User Category
In this field, you can specify the user category for this user.
Last Sign-on
This field shows the last time an existing user signed into the system. It will be blank for new users.


Once you have entered all the information, click on the Save button to add the new user account or save the changes made to an existing user account.

Once a new user account has been created, the user’s rights have to be set in the input models, or else the user will not be able to view, edit or create any records.
