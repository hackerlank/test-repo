Termbases and Projects
 Previous Top Next

The Termbases tab can be used to create and manage Termbases and projects.

Use the drop-down list on the left-hand side to select an existing Termbase or click on the New Termbase... button to create a new Termbase. The name of an existing Termbase can also be changed in this tab.

Once a Termbase has been selected, use the drop-down list on the right-hand side to select an existing project or use the New Project... button to create a new project. The name of an existing project can also be changed.
